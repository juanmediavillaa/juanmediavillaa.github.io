Bitcoin Trading Bot

Utilizing VSCode as the coding platform and Python as the programming language, I developed a program incorporating various standard technical analysis tools. These tools determined buy and sell signals within the market. To execute this program with real capital, I integrated the Binance API. To ensure uninterrupted operation, I maintained Google Cloud Servers running around the clock, enabling the continuous functionality of the trading bot.
